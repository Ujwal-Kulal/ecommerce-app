# Ecommerce App (Full Stack)

A full-stack ecommerce learning project with:
- Node.js + Express backend API
- React + Vite frontend (TypeScript version)
- React + Vite frontend (JavaScript version)

The app supports product listing, cart updates, checkout, orders, and tracking flow.

---

## Project Structure

- `ecommerce-backend` → Express API + data/models/routes + static image hosting
- `ecommerce-project` → React + TypeScript frontend
- `ecommerce-project-js` → React + JavaScript frontend
- `starting-code` → starter/reference files

---

## Tech Stack

- **Backend:** Node.js, Express, Sequelize, sql.js
- **Frontend:** React 19, Vite, Axios, React Router
- **Tooling:** ESLint, Vitest, Testing Library

---

## Prerequisites

- Node.js (v22+ recommended)
- npm
